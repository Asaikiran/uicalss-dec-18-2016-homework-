[1]how to resolve merge conflicts?
-->To resolve a merge conflict caused by competing changes to a file, where a person deletes a file in one branch and another person edits the same file, you must choose whether to delete or keep the removed file in a new commit
For example, if you edited a file, such as README.md, and another person removed the same file in another branch in the same Git repository, you'll get a merge conflict error when you try to merge these branches. You must resolve this merge conflict with a new commit before you can merge these branches.

[4]what is tag and element in html?
-->HTML tags are the hidden keywords within a web page that define how the browser must format and display the content.
Most tags must have two parts, an opening and a closing part. For example, <html> is the opening tag and </html> is the closing tag. Note that the closing tag has the same text as the opening tag, but has an additional forward-slash ( / ) character
An HTML element usually consists of a start tag and end tag, with the content inserted in between:

<tagname>Content goes here...</tagname>
The HTML element is everything from the start tag to the end tag:

<p>My first paragraph.</p>

[5]what is an attribute in html?

Attributes usually come in name/value pairs like: name="value"
